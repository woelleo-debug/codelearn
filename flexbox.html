Der komplette Flexbox Guide

Flexbox ist eines der wichtigsten Layout-Tools im modernen CSS. Lerne alle Properties mit visuellen Beispielen und echtem Code. Nach diesem Modul kannst du responsive Layouts mit Flexbox erstellen.

📖 Was ist Flexbox?

Flexbox (Flexible Box Layout) ist ein CSS Layout-Modul, das es dir ermöglicht, Elemente in einem Container auf flexible und effiziente Weise auszurichten und zu verteilen.

Warum Flexbox?

  • Responsive Layouts ohne Media Queries (oft)
  • Elemente verteilen sich automatisch
  • Perfekte Zentrierung möglich
  • Flexible Reihenfolge ohne HTML ändern
  • Einfach und intuitiv
💡 Merke: Flexbox funktioniert auf einer Achse (horizontal oder vertikal). Für 2D-Layouts nutze CSS Grid.

🖥️ Display: Flex

Um Flexbox zu nutzen, setze display: flex auf einem Container:

HTML & CSS 
<!-- HTML -->
<div class="flex-container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
</div>

<!-- CSS -->
.flex-container {
  display: flex;              /* Aktiviert Flexbox */
  gap: 10px;                  /* Abstand zwischen Kindern */
  border: 2px solid #333;
  padding: 10px;
}

.item {
  background: #3b82f6;
  padding: 20px;
  color: white;
  text-align: center;
  border-radius: 6px;
}
Ergebnis: Die 3 Elemente werden nebeneinander angeordnet (horizontal), nicht untereinander.

↕️ Flex-Direction

Mit flex-direction legst du fest, in welche Richtung Elemente angeordnet werden:

CSS 
.container {
  display: flex;
  
  /* row (Standard): Horizontal, von links nach rechts */
  flex-direction: row;
  
  /* column: Vertikal, von oben nach unten */
  flex-direction: column;
  
  /* row-reverse: Horizontal, von rechts nach links */
  flex-direction: row-reverse;
  
  /* column-reverse: Vertikal, von unten nach oben */
  flex-direction: column-reverse;
}
Wert Beschreibung Hauptachse
row Standard – horizontal Horizontal (links → rechts)
column Vertikal Vertikal (oben → unten)
row-reverse Horizontal rückwärts Horizontal (rechts → links)
column-reverse Vertikal rückwärts Vertikal (unten → oben)

←→ Justify-Content (Hauptachse)

Mit justify-content ordnest du Elemente entlang der Hauptachse an:

CSS 
.container {
  display: flex;
  width: 500px;
  border: 2px solid #333;
  
  /* flex-start (Standard): Alle Items am Anfang */
  justify-content: flex-start;
  
  /* flex-end: Alle Items am Ende */
  justify-content: flex-end;
  
  /* center: Items zentriert */
  justify-content: center;
  
  /* space-between: Gleichmäßig verteilt (erste/letzte am Rand) */
  justify-content: space-between;
  
  /* space-around: Gleichmäßig mit Abstand rundherum */
  justify-content: space-around;
  
  /* space-evenly: Perfekt gleichmäßig */
  justify-content: space-evenly;
}

Visualisierung:

flex-start: [A][B][C]___

flex-end: ___[A][B][C]

center: __[A][B][C]__

space-between: [A]__[B]__[C]

space-around: _[A]_[B]_[C]_

space-evenly: _[A]_[B]_[C]_

↑↓ Align-Items (Nebenachse)

Mit align-items ordnest du Elemente entlang der Nebenachse an (senkrecht zur Hauptachse):

CSS 
.container {
  display: flex;
  height: 200px;    /* Wichtig für align-items! */
  border: 2px solid #333;
  
  /* stretch (Standard): Items strecken die ganze Höhe */
  align-items: stretch;
  
  /* flex-start: Items oben */
  align-items: flex-start;
  
  /* flex-end: Items unten */
  align-items: flex-end;
  
  /* center: Items zentriert (vertikal!) */
  align-items: center;
  
  /* baseline: Items auf Baseline ausrichten */
  align-items: baseline;
}
✅ Profi-Tipp: Verwende display: flex; justify-content: center; align-items: center; um Elemente perfekt zu zentrieren!

↪️ Align-Self (Einzelne Items)

Mit align-self kannst du ein einzelnes Item anders ausrichten als die anderen:

CSS 
.container {
  display: flex;
  align-items: center;    /* Standard für alle */
  height: 200px;
}

.item-1 {
  align-self: flex-start;   /* Dieses Item oben */
}

.item-2 {
  align-self: center;       /* Bleibt zentriert (Standard) */
}

.item-3 {
  align-self: flex-end;     /* Dieses Item unten */
}

.item-4 {
  align-self: stretch;      /* Füllt die ganze Höhe */
}

📏 Flex-Wrap

Standardmäßig quetschen sich Flex-Items in eine Zeile. Mit flex-wrap kannst du Umbruch erlauben:

CSS 
.container {
  display: flex;
  width: 300px;    /* Nicht breit genug für alle Items */
  
  /* nowrap (Standard): Items quetschen sich in eine Zeile */
  flex-wrap: nowrap;
  
  /* wrap: Items umbrechen zu nächster Zeile */
  flex-wrap: wrap;
  
  /* wrap-reverse: Items umbrechen, aber rückwärts */
  flex-wrap: wrap-reverse;
}

Beispiel: Wenn Container 300px breit und Items je 150px sind:

  • nowrap: Alle 3 Items werden gequetscht in eine Zeile
  • wrap: 2 Items in erste Zeile, 1 Item in zweite Zeile

⚙️ Flex, Flex-Grow, Flex-Shrink, Flex-Basis

Diese Properties kontrollieren, wie Flex-Items Platz einnehmen:

CSS 
.container {
  display: flex;
  gap: 10px;
}

/* flex-basis: Initiale Größe vor flex-grow/shrink */
.item-1 {
  flex-basis: 100px;    /* Startet mit 100px */
}

/* flex-grow: Wie viel zusätzlichen Platz nimmt es ein? (0-1+) */
.item-2 {
  flex-grow: 1;         /* Nimmt 1 Teil zusätzlichen Platz */
}

.item-3 {
  flex-grow: 2;         /* Nimmt 2 Teile zusätzlichen Platz (doppelt!) */
}

/* flex-shrink: Wie viel schrumpft es bei wenig Platz? */
.item-4 {
  flex-shrink: 0;       /* Schrumpft nicht */
}

/* Kurzform: flex = flex-grow flex-shrink flex-basis */
.item {
  flex: 1 0 100px;      /* grow=1, shrink=0, basis=100px */
  flex: 1;              /* Gleichmäßig wachsen (grow=1) */
}
Property Wert Beschreibung
flex-grow 0-∞ Anteil beim Vergrößern (0 = nicht vergrößern)
flex-shrink 0-∞ Anteil beim Verkleinern (0 = nicht verkleinern)
flex-basis auto, %,px Basis-Größe vor grow/shrink

➕ Gap – Abstände zwischen Items

Mit gap definierst du automatische Abstände zwischen Flex-Items:

CSS 
.container {
  display: flex;
  gap: 15px;              /* 15px Abstand überall */
  
  /* gap: row-gap column-gap */
  gap: 10px 20px;         /* 10px vertikal, 20px horizontal */
  
  row-gap: 10px;          /* Abstand zwischen Zeilen */
  column-gap: 20px;       /* Abstand zwischen Spalten */
}

.item {
  padding: 10px;
  background: #3b82f6;
}
💡 Vorteil: gap ist besser als margin, weil es keinen Abstand am Anfang/Ende erzeugt.

🎯 Praktische Beispiele

Beispiel 1: Navigationsleiste

HTML & CSS 
<nav class="navbar">
  <a href="/" class="logo">Logo</a>
  <div class="nav-links">
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Contact</a>
  </div>
  <button class="btn">Sign In</button>
</nav>

<style>
.navbar {
  display: flex;
  justify-content: space-between;   /* Logo links, Content rechts */
  align-items: center;               /* Alles vertikal zentriert */
  gap: 20px;
  padding: 15px 30px;
  background: #333;
}

.nav-links {
  display: flex;
  gap: 20px;
}

.nav-links a {
  color: white;
  text-decoration: none;
}
</style>

Beispiel 2: 3-Spalten Layout

HTML & CSS 
<div class="container">
  <aside class="sidebar">Sidebar</aside>
  <main class="content">Hauptinhalt</main>
  <aside class="sidebar-right">Sidebar</aside>
</div>

<style>
.container {
  display: flex;
  gap: 20px;
}

.sidebar {
  width: 200px;          /* Feste Breite */
}

.sidebar-right {
  width: 200px;
}

.content {
  flex: 1;               /* Nimmt restlichen Platz */
}

/* Mobile: Stack vertikal */
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
  
  .sidebar, .sidebar-right {
    width: auto;
  }
}
</style>

Beispiel 3: Karten-Grid

HTML & CSS 
<div class="cards">
  <div class="card">Karte 1</div>
  <div class="card">Karte 2</div>
  <div class="card">Karte 3</div>
</div>

<style>
.cards {
  display: flex;
  gap: 20px;
  flex-wrap: wrap;       /* Umbruch erlauben */
  justify-content: center;
}

.card {
  flex: 0 1 calc(33.333% - 14px);  /* 3 pro Zeile */
  min-height: 200px;
  background: #3b82f6;
  padding: 20px;
  border-radius: 8px;
}

/* 2 pro Zeile auf Tablet */
@media (max-width: 1024px) {
  .card {
    flex: 0 1 calc(50% - 10px);
  }
}

/* 1 pro Zeile auf Mobile */
@media (max-width: 640px) {
  .card {
    flex: 0 1 100%;
  }
}
</style>

🎓 Zusammenfassung

Wichtigste Flexbox-Properties:

  • display: flex – Aktiviert Flexbox
  • flex-direction – Hauptachsen-Richtung
  • justify-content – Ausrichtung auf Hauptachse
  • align-items – Ausrichtung auf Nebenachse
  • flex-wrap – Umbruch erlauben
  • gap – Abstände zwischen Items
  • flex: 1 – Item wächst flexibel
