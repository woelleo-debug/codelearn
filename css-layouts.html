CSS Layouts

Lerne die Grundlagen moderner CSS-Layouts: Box Model, Display-Typen, Positioning und Overflow. Diese Konzepte sind die Basis für Flexbox und Grid.

📖 Layout-Fundamente

In CSS gibt es verschiedene Wege, um Elemente auf einer Seite anzuordnen. Die wichtigsten Konzepte sind:

  • Box Model: Wie Margin, Border, Padding und Content zusammenhängen
  • Display: Wie Elemente in der normalen Dokumentenfluss angeordnet werden
  • Positioning: Absolute und relative Positionierung von Elementen
  • Flexbox & Grid: Moderne Layout-Methoden (siehe spätere Module)
💡 Tipp: Moderne Websites verwenden hauptsächlich Flexbox und CSS Grid für Layouts. Aber die Grundlagen hier sind immer noch wichtig!

📦 Das Box Model

Alle HTML-Elemente werden in CSS als rechteckige Boxen dargestellt. Das Box Model beschreibt, aus welchen Teilen diese Boxen bestehen:

CSS 
/* Das Box Model von außen nach innen: */
.box {
  /* Margin: Abstand nach außen (transparent) */
  margin: 20px;
  
  /* Border: Linie um die Box */
  border: 2px solid #333;
  
  /* Padding: Abstand nach innen (hat Background-Farbe) */
  padding: 15px;
  
  /* Content: Der eigentliche Inhalt */
  width: 200px;
  height: 100px;
  background-color: #3b82f6;
}

/* Kurz-Schreibweisen */
margin: 10px 20px;        /* oben/unten 10px, links/rechts 20px */
margin: 10px 20px 30px 40px; /* oben, rechts, unten, links (clockwise) */
padding: 15px;            /* überall 15px */

Gesamtgröße berechnen:

Breite = Margin-left + Border + Padding + Width + Padding + Border + Margin-right

📌 Inline vs Block-Elemente

HTML-Elemente verhalten sich standardmäßig entweder als Block- oder Inline-Elemente:

Block-Elemente Inline-Elemente
Nehmen die ganze Breite ein Nur so breit wie nötig
Margin und Padding wirken Vertikales Margin/Padding wirkt nicht
Beispiele: <div>, <p>, <h1-h6>, <ul> Beispiele: <span>, <a>, <button>, <img>
CSS 
/* Display-Eigenschaft ändern */
span {
  display: block;           /* Inline-Element zu Block */
}

div {
  display: inline;          /* Block-Element zu Inline */
}

a {
  display: inline-block;    /* Beste von beiden Welten! */
  padding: 10px 15px;       /* Jetzt wirkt Padding! */
  margin: 5px;              /* Und Margin auch! */
}

🖥️ Die Display-Eigenschaft

Mit der display-Eigenschaft kontrollierst du, wie ein Element in den Dokumentenfluss passt:

CSS 
/* Wichtige Display-Werte */

/* block: Vollständige Zeile */
.block { display: block; }

/* inline: Neben- bzw. Fließtext */
.inline { display: inline; }

/* inline-block: Block in Inline-Flow */
.inline-block { display: inline-block; }

/* none: Element wird nicht angezeigt */
.hidden { display: none; }

/* flex: Flexbox Layout (sehr wichtig!) */
.container { display: flex; }

/* grid: CSS Grid Layout (sehr wichtig!) */
.grid { display: grid; }

/* table: Wie HTML-Tabelle */
.table { display: table; }

📍 Positioning

Mit positioning kannst du Elemente von ihrem normalen Platz im Dokumentenfluss wegschieben:

CSS 
/* Position-Werte */

/* static (Standard): Normal im Flow */
.static { position: static; }

/* relative: Relativ zur ursprünglichen Position */
.relative { 
  position: relative;
  top: 20px;      /* 20px nach unten */
  left: 30px;     /* 30px nach rechts */
}

/* absolute: Relativ zum nächsten positioned Eltern-Element */
.container {
  position: relative;    /* Anchor für absolute Kinder */
  width: 200px;
  height: 200px;
}

.absolute {
  position: absolute;
  top: 10px;             /* 10px vom oben des Containers */
  left: 10px;            /* 10px vom linken Ende des Containers */
  width: 50px;
  height: 50px;
}

/* fixed: Bleibt beim Scrollen an Position */
.header {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;           /* Über der ganzen Seite */
}

/* sticky: Wechselt zwischen relative und fixed */
.sticky-header {
  position: sticky;
  top: 0;                /* Fixiert oben wenn Seite scrollt */
}

🎯 Z-Index und Stacking-Kontext

Z-Index kontrolliert, welche Elemente oben oder unten liegen, wenn sie sich überlappen:

CSS 
/* Z-Index funktioniert nur mit positionierten Elementen */
.box-1 {
  position: relative;
  z-index: 10;    /* Liegt oben */
  background: blue;
}

.box-2 {
  position: relative;
  z-index: 5;     /* Liegt darunter */
  background: red;
}

.box-3 {
  position: relative;
  z-index: 1;     /* Liegt ganz unten */
  background: green;
}

/* Höherer Z-Index = weiter oben */
/* Standard Z-Index = auto (0) */

/* Wichtig: Z-Index nur zwischen positioned Elementen */
.static-element {
  position: static;
  z-index: 9999;  /* Funktioniert NICHT! */
}

⊕ Overflow – Was passiert bei zu viel Inhalt?

Wenn Inhalt größer als sein Container ist, kannst du mit overflow kontrollieren, was passiert:

CSS 
/* Overflow-Werte */

/* visible (Standard): Inhalt überläuft den Container */
.visible {
  overflow: visible;
  width: 200px;
  height: 100px;
}

/* hidden: Überlaufender Inhalt wird abgeschnitten */
.hidden {
  overflow: hidden;
  width: 200px;
  height: 100px;
}

/* scroll: Scrollbars werden hinzugefügt */
.scroll {
  overflow: scroll;
  width: 200px;
  height: 100px;
}

/* auto: Scrollbars nur wenn nötig */
.auto {
  overflow: auto;
  width: 200px;
  height: 100px;
}

/* Separate Kontrolle für x und y */
.box {
  overflow-x: hidden;     /* Horizontal abschneiden */
  overflow-y: auto;       /* Vertikal scrollbar nur wenn nötig */
}

🔲 Container und Best Practices

Ein wichtiges Konzept ist der "Container" – ein äußeres Div, das andere Elemente gruppiert und begrenzt:

HTML & CSS 
<!-- HTML -->
<div class="container">
  <div class="content">
    <h1>Titel</h1>
    <p>Inhalt...</p>
  </div>
</div>

<!-- CSS -->
.container {
  max-width: 1200px;     /* Maximale Breite */
  margin: 0 auto;        /* Horizontal zentriert */
  padding: 40px 20px;    /* Abstand innen */
}

.content {
  width: 100%;           /* Füllt den Container */
}
✅ Best Practice: Verwende Container um deinen Inhalt zu begrenzen und Abstände konsistent zu halten. Das ist die Basis vieler moderner Websites.

🎯 Praktisches Beispiel: Komplexes Layout

HTML & CSS 
<!-- HTML -->
<body>
  <header class="header">Navigation</header>
  <main class="main-content">
    <aside class="sidebar">Sidebar</aside>
    <section class="content">Hauptinhalt</section>
  </main>
  <footer class="footer">Footer</footer>
</body>

<!-- CSS -->
* { box-sizing: border-box; }

.header {
  position: sticky;
  top: 0;
  background: #333;
  padding: 20px;
  z-index: 100;
}

.main-content {
  display: flex;
  max-width: 1200px;
  margin: 0 auto;
}

.sidebar {
  width: 250px;
  padding: 20px;
  overflow-y: auto;
}

.content {
  flex: 1;
  padding: 40px;
}

.footer {
  background: #222;
  padding: 40px;
  text-align: center;
}
