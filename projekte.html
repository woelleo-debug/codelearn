💼 Projekt 2: Portfolio Website

      
Zeige deine Projekte in einer modernen Portfolio Website. Perfekt für dein eigenes Portfolio!

      

        💡 Lernziele: CSS Grid, Bilder responsiv machen, Hover-Effekte, Layout-Techniken
      

      

        

          CSS
          
        

        
/* Portfolio Grid */
.portfolio-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
  margin-top: 30px;
}

.portfolio-item {
  position: relative;
  overflow: hidden;
  border-radius: 10px;
  height: 250px;
}

.portfolio-item img {
  width: 100%;
  height: 100%;
  object-fit: cover;
  transition: transform 0.3s;
}

.portfolio-item:hover img {
  transform: scale(1.1);
}

.portfolio-item-overlay {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;

  display: flex;
  align-items: center;
  justify-content: center;
  flex-direction: column;
  opacity: 0;
  transition: opacity 0.3s;
}

.portfolio-item:hover .portfolio-item-overlay {
  opacity: 1;
}

.portfolio-item-overlay h3 {
  color: white;

🖼️ Projekt 6: Responsive Bildergalerie

      
Eine Galerie, die sich automatisch an die Bildschirmgröße anpasst.

      

        

          CSS
          
        

        
.gallery {
  display: grid;
  gap: 15px;
  margin-top: 30px;
}

/* Desktop: 4 Spalten */
@media (min-width: 1200px) {
  .gallery {
    grid-template-columns: repeat(4, 1fr);
  }
}

/* Tablet: 3 Spalten */
@media (min-width: 768px) and (max-width: 1199px) {
  .gallery {
    grid-template-columns: repeat(3, 1fr);
  }
}

/* Smartphone: 2 Spalten */
@media (max-width: 767px) {
  .gallery {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Extra Small: 1 Spalte */
@media (max-width: 480px) {
  .gallery {
    grid-template-columns: 1fr;
  }
}

.gallery-item {
  position: relative;
  overflow: hidden;
  aspect-ratio: 1 / 1;
  border-radius: 8px;
  cursor: pointer;
}

.gallery-item img {
  width: 100%;
  height: 100%;
  object-fit: cover;
  transition: transform 0.3s;
}

.gallery-item:hover img {
  transform: scale(1.1);
}
