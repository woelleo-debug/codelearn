CSS Animationen

Lerne, wie du elegante Animationen und Übergänge mit reinem CSS erstellst. Vom einfachen Hover-Effekt bis zu komplexen Keyframe-Animationen.

📖 Animationen in CSS

CSS Animationen bringen deine Website zum Leben! Es gibt zwei Hauptarten:

  • Transitions: Sanfte Übergänge zwischen Zuständen (einfach)
  • Keyframe Animations: Komplexe, mehrstufige Animationen (flexibel)
💡 Performance-Tipp: CSS-Animationen laufen auf der GPU und sind viel schneller als JavaScript-Animationen!

🎬 CSS Transitions

Transitions erzeugen sanfte Übergänge zwischen CSS-Eigenschafts-Werten:

HTML & CSS 
<!-- HTML -->
<div class="button">Hover mich!</div>

<!-- CSS -->
.button {
  background-color: #3b82f6;
  padding: 15px 30px;
  border: none;
  cursor: pointer;
  
  /* Transition: Eigenschaft Dauer Timing-Funktion */
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: #ef4444;
}

Transition Syntax:

CSS 
.element {
  /* Einzelne Eigenschaft */
  transition: background-color 0.3s ease;
  
  /* Mehrere Eigenschaften */
  transition: background-color 0.3s ease, color 0.3s ease, transform 0.3s ease;
  
  /* Alle Properties animieren */
  transition: all 0.3s ease;
  
  /* Mit Verzögerung */
  transition: all 0.3s ease 0.1s;  /* 0.1s Verzögerung */
  
  /* Ausführlich */
  transition-property: all;
  transition-duration: 0.3s;
  transition-timing-function: ease;
  transition-delay: 0s;
}

🎨 @keyframes Animationen

Mit @keyframes kannst du komplexe, mehrstufige Animationen definieren:

CSS 
/* Definiere eine Animation */
@keyframes slide-in {
  0% {
    opacity: 0;
    transform: translateX(-100px);
  }
  
  50% {
    opacity: 0.5;
  }
  
  100% {
    opacity: 1;
    transform: translateX(0);
  }
}

/* Wende die Animation an */
.box {
  animation: slide-in 0.5s ease;
}

/* Alternativ: from/to */
@keyframes fade {
  from {
    opacity: 0;
  }
  to {
    opacity: 1;
  }
}

⚙️ Animation Properties

Diese Properties kontrollieren, wie Keyframe-Animationen ablaufen:

Property Beschreibung Werte
animation-name Name der @keyframes slide-in, fade
animation-duration Dauer der Animation 0.5s, 2000ms
animation-timing-function Beschleunigung ease, linear, ease-in-out
animation-delay Verzögerung vor Start 0s, 0.5s
animation-iteration-count Wie oft wiederholen? 1, 3, infinite
animation-direction Richtung normal, reverse, alternate
animation-fill-mode Style vor/nach Animation forwards, backwards, both
CSS 
.animated-box {
  animation-name: bounce;
  animation-duration: 1s;
  animation-timing-function: ease;
  animation-iteration-count: infinite;    /* Endlos */
  animation-direction: alternate;         /* Vorwärts/Rückwärts */
  
  /* Kurzform */
  animation: bounce 1s ease infinite alternate;
}

⏱️ Timing Functions

Timing Functions definieren, wie schnell die Animation läuft:

Funktion Beschreibung Effekt
linear Konstante Geschwindigkeit Gleichmäßig
ease Standard – langsam, schnell, langsam Natural Feel
ease-in Langsamer Start Startet langsam
ease-out Schneller Start, langsames Ende Endet langsam
ease-in-out Langsam, schnell, langsam Sehr smooth

Custom Timing mit cubic-bezier():

cubic-bezier(0.25, 0.46, 0.45, 0.94)

👆 Interaktive Hover-Effekte

Kombiniere Transitions mit :hover für schöne Interaktionen:

CSS 
/* Einfache Farb-Änderung */
.button {
  transition: all 0.3s ease;
}

.button:hover {
  background-color: red;
  transform: scale(1.05);
  box-shadow: 0 5px 15px rgba(0,0,0,0.3);
}

/* Aufzählungs-Element */
.item {
  transition: transform 0.3s ease;
}

.item:hover {
  transform: translateY(-5px);
}

/* Icon Rotation */
.icon {
  transition: transform 0.3s ease;
}

.icon:hover {
  transform: rotate(180deg);
}

/* Unterschrift Unterstrich */
a {
  position: relative;
  text-decoration: none;
  transition: color 0.3s ease;
}

a::after {
  content: '';
  position: absolute;
  bottom: 0;
  left: 0;
  width: 0;
  height: 2px;
  background: #3b82f6;
  transition: width 0.3s ease;
}

a:hover::after {
  width: 100%;
}

⏳ Loading Animationen

Häufig verwendete Loading-Effekte:

HTML & CSS 
<!-- Spinner -->
<div class="spinner"></div>

<style>
@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

.spinner {
  width: 40px;
  height: 40px;
  border: 4px solid rgba(255,255,255,0.3);
  border-top: 4px solid #3b82f6;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}
</style>

<!-- Pulsing Dot -->
<div class="pulse"></div>

<style>
@keyframes pulse {
  0% { opacity: 1; transform: scale(1); }
  50% { opacity: 0.5; transform: scale(1.2); }
  100% { opacity: 1; transform: scale(1); }
}

.pulse {
  width: 20px;
  height: 20px;
  background: #3b82f6;
  border-radius: 50%;
  animation: pulse 2s ease-in-out infinite;
}
</style>

🎯 Praktische Animationsmuster

Fade-In beim Seitenload

CSS 
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.section {
  animation: fadeIn 0.6s ease-out;
}

.section:nth-child(1) { animation-delay: 0.1s; }
.section:nth-child(2) { animation-delay: 0.2s; }
.section:nth-child(3) { animation-delay: 0.3s; }

Shake Effekt

CSS 
@keyframes shake {
  0%, 100% { transform: translateX(0); }
  25% { transform: translateX(-10px); }
  75% { transform: translateX(10px); }
}

.error:hover {
  animation: shake 0.3s ease-in-out;
}

Slide Down Navigation

CSS 
@keyframes slideDown {
  from {
    opacity: 0;
    transform: translateY(-100%);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

nav {
  animation: slideDown 0.5s ease-out;
}

✅ Best Practices für Animationen

  • Moderation: Nicht zu viele Animationen – das wirkt überladen
  • Performance: Nutze nur animierte CSS-Properties (transform, opacity)
  • Duration: Meistens 0.3-0.5 Sekunden für Interactions
  • User Experience: Respektiere prefers-reduced-motion für Accessibility
  • Konsistenz: Nutze ähnliche Timings und Effekte auf der ganzen Seite
