Der komplette CSS Grid Guide

CSS Grid ist das modernste Layout-Tool für 2D-Layouts. Lerne, wie du mit Zeilen und Spalten komplexe Seiten-Layouts erstellst – mit visuellen Beispielen und praxisnahem Code.

📖 Was ist CSS Grid?

CSS Grid ist ein 2D-Layout-System, das dir ermöglicht, Layouts mit Zeilen UND Spalten zu erstellen. Im Gegensatz zu Flexbox (1D) kannst du mit Grid Elemente in beide Richtungen kontrollieren.

Wann nutze ich Grid?

  • Komplexe Website-Layouts (Header, Sidebar, Content, Footer)
  • Dashboard-Layouts mit mehreren Spalten
  • Responsive Galerien
  • Magazine-ähnliche Designs
💡 Flexbox vs Grid: Flexbox für 1D (Reihen oder Spalten), Grid für 2D (Reihen UND Spalten).

🖥️ Display: Grid

Um CSS Grid zu nutzen, setze display: grid auf einem Container:

HTML & CSS 
<!-- HTML -->
<div class="grid-container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
</div>

<!-- CSS -->
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;  /* 3 Spalten */
  grid-template-rows: auto auto;       /* 2 Zeilen */
  gap: 15px;                           /* Abstand */
}

.item {
  background: #3b82f6;
  padding: 40px;
  text-align: center;
  border-radius: 6px;
}
Ergebnis: 6 Elemente in einem 3×2 Grid (3 Spalten, 2 Zeilen)

📊 Grid-Template-Columns

Mit grid-template-columns definierst du die Spalten des Grids:

CSS 
.container {
  display: grid;
  
  /* Feste Breiten */
  grid-template-columns: 200px 200px 200px;
  
  /* Flexible Breiten mit fr (Bruchteile) */
  grid-template-columns: 1fr 1fr 1fr;   /* 3 gleiche Spalten */
  grid-template-columns: 2fr 1fr;       /* Erste doppelt so breit */
  
  /* Mischung */
  grid-template-columns: 200px 1fr 150px;   /* Fest, flexibel, fest */
  
  /* Auto (passt sich Inhalt an) */
  grid-template-columns: auto 1fr auto;
  
  /* repeat() Funktion */
  grid-template-columns: repeat(3, 1fr);     /* 3 gleiche Spalten */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* Responsive! */
  
  /* Minmax */
  grid-template-columns: minmax(200px, 1fr) 200px;
}
Wert Beschreibung
200px Feste Breite von 200 Pixeln
1fr Ein Bruchteil des verfügbaren Platzes
auto Passt sich dem Inhalt an
repeat(3, 1fr) 3 Spalten mit je 1fr
minmax(200px, 1fr) Mindestens 200px, maximal 1fr

📊 Grid-Template-Rows

Mit grid-template-rows definierst du die Höhen der Zeilen:

CSS 
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  
  /* Feste Höhen */
  grid-template-rows: 100px 150px 200px;
  
  /* Auto (Höhe dem Inhalt anpassen) */
  grid-template-rows: auto auto auto;
  
  /* Flexibel */
  grid-template-rows: 1fr 1fr 1fr;    /* Gleiche Höhen */
  
  /* Mischung */
  grid-template-rows: 100px 1fr 100px;
  
  /* Repeat */
  grid-template-rows: repeat(2, auto);
}

➕ Gap – Abstände im Grid

Mit gap definierst du Abstände zwischen Grid-Items:

CSS 
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  
  /* Gleicher Abstand überall */
  gap: 20px;
  
  /* Separate row-gap und column-gap */
  gap: 10px 20px;        /* Zeilen-Abstand, Spalten-Abstand */
  row-gap: 10px;         /* Nur zwischen Zeilen */
  column-gap: 20px;      /* Nur zwischen Spalten */
}

⚙️ Grid-Auto-Columns & Grid-Auto-Rows

Diese Properties definieren automatisch erzeugte Spalten und Zeilen:

CSS 
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* Nur 3 Spalten definiert */
  grid-template-rows: repeat(2, 100px);   /* Nur 2 Zeilen definiert */
  
  /* Wenn mehr als 6 Items: neue Zeilen automatisch erzeugen */
  grid-auto-rows: 100px;    /* Jede neue Zeile 100px hoch */
  
  /* Neue Spalten automatisch erzeugen */
  grid-auto-columns: 150px;
}

/* Beispiel: grid-auto-flow */
.container {
  grid-auto-flow: row;        /* Standard: von links nach rechts, dann unten */
  grid-auto-flow: column;     /* Von oben nach unten, dann rechts */
  grid-auto-flow: dense;      /* Lücken füllen */
}

🎯 Grid-Template-Areas

Mit grid-template-areas kannst du Bereiche wie ein ASCII-Art Layout definieren:

HTML & CSS 
<!-- HTML -->
<div class="layout">
  <header class="header">Header</header>
  <nav class="nav">Navigation</nav>
  <main class="main">Main Content</main>
  <aside class="sidebar">Sidebar</aside>
  <footer class="footer">Footer</footer>
</div>

<!-- CSS -->
.layout {
  display: grid;
  grid-template-areas:
    "header   header   header"
    "nav      main     sidebar"
    "nav      main     sidebar"
    "footer   footer   footer";
  grid-template-columns: 150px 1fr 200px;
  grid-template-rows: auto 1fr auto;
  gap: 20px;
}

.header {
  grid-area: header;
  background: #333;
}

.nav {
  grid-area: nav;
  background: #444;
}

.main {
  grid-area: main;
}

.sidebar {
  grid-area: sidebar;
}

.footer {
  grid-area: footer;
  background: #333;
}

📍 Grid-Column & Grid-Row (Platzierung)

Mit diesen Properties kannst du Items auf spezifische Grid-Positionen platzieren:

HTML & CSS 
<!-- HTML -->
<div class="container">
  <div class="item-1">1</div>
  <div class="item-2">2</div>
  <div class="item-3">3</div>
</div>

<!-- CSS -->
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(3, 100px);
}

/* Item 1 von Spalte 1 bis 3 */
.item-1 {
  grid-column: 1 / 3;    /* Startet Spalte 1, endet Spalte 3 */
  grid-row: 1 / 2;       /* Zeile 1 */
}

/* Item 2 von Zeile 1 bis 3 */
.item-2 {
  grid-column: 3 / 5;    /* Spalten 3-5 */
  grid-row: 1 / 3;       /* Zeilen 1-3 */
}

/* Kurzform: grid-column: start / end */
.item-3 {
  grid-column: 1 / -1;   /* Alle Spalten (-1 = letzter) */
  grid-row: 3;           /* Zeile 3 */
}

🎯 Praktische Beispiele

Beispiel 1: Website-Layout

CSS 
body {
  display: grid;
  grid-template-areas:
    "header"
    "nav"
    "main"
    "footer";
  grid-template-columns: 1fr;
  min-height: 100vh;
}

header { grid-area: header; }
nav { grid-area: nav; }
main { grid-area: main; flex: 1; }
footer { grid-area: footer; }

/* Desktop: Header über alles, Sidebar links */
@media (min-width: 1024px) {
  body {
    grid-template-areas:
      "header header"
      "nav main"
      "footer footer";
    grid-template-columns: 250px 1fr;
  }
}

Beispiel 2: Responsive Galerie

CSS 
.gallery {
  display: grid;
  gap: 15px;
}

/* Desktop: Auto-responsive Spalten */
@media (min-width: 1024px) {
  .gallery {
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  }
}

/* Tablet: 2 Spalten */
@media (min-width: 768px) and (max-width: 1023px) {
  .gallery {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Mobile: 1 Spalte */
@media (max-width: 767px) {
  .gallery {
    grid-template-columns: 1fr;
  }
}

Beispiel 3: Dashboard

CSS 
.dashboard {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 20px;
  padding: 20px;
}

.card {
  background: white;
  border-radius: 8px;
  padding: 20px;
}

/* Verschiedene Größen */
.card-large {
  grid-column: span 6;    /* 6 von 12 Spalten */
}

.card-medium {
  grid-column: span 4;    /* 4 von 12 Spalten */
}

.card-small {
  grid-column: span 2;    /* 2 von 12 Spalten */
}

@media (max-width: 768px) {
  .card-large, .card-medium, .card-small {
    grid-column: span 12;  /* Alle full-width auf Mobile */
  }
}

🎓 Zusammenfassung

Wichtigste Grid-Properties:

  • display: grid – Aktiviert CSS Grid
  • grid-template-columns – Spalten definieren
  • grid-template-rows – Zeilen definieren
  • gap – Abstände zwischen Items
  • grid-template-areas – Benannte Bereiche
  • grid-column / grid-row – Item-Platzierung
  • repeat(), minmax(), auto-fit – Responsive Funktionen
